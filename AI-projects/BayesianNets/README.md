# Bayesian Networks

A probabilistic AI project implementing Bayesian Network inference and Naive Bayes classification from scratch in Python. The project performs exact inference using Variable Elimination and applies probabilistic modeling to predict diabetes diagnoses from patient data.

Developed for CMPSC 442: Artificial Intelligence at Penn State.

## Features

- Variable Elimination for exact Bayesian Network inference
- Factor restriction, joining, and variable elimination
- Probability distribution normalization
- Naive Bayes classifier for diabetes prediction
- Conditional Probability Table (CPT) generation from training data
- Model evaluation using prediction accuracy on a test dataset

## Main Components

- Bayesian Network factor operations
- Variable Elimination algorithm
- Conditional Probability Table construction
- Naive Bayes model training
- Probabilistic inference and prediction
- Classification accuracy evaluation

## File Structure

- `solution_q1.py` – Bayesian Network inference using Variable Elimination (my work)
- `solution_q2.py` – Naive Bayes classifier and diabetes prediction model (my work)
- `Naive-Bayes-Classification-Data.csv` – Dataset used for model training and evaluation

## How to Run

```bash
python solution_q1.py
python solution_q2.py
```
