# River Crossing Search Algorithms

A Python implementation of classical AI search algorithms to solve the Missionaries and Cannibals river crossing puzzle. The project implements uninformed and informed search techniques, compares their performance, and evaluates custom admissible heuristics.

Developed for CMPSC 442: Artificial Intelligence at Penn State.

## Features

- Depth-First Search (DFS) and Breadth-First Search (BFS)
- Uniform Cost Search (UCS) with multiple cost models
- A* Search with custom admissible heuristics
- State validation and successor generation
- Solution path reconstruction and node expansion tracking
- Comparison of search strategies based on cost and efficiency

## Main Components

- State representation and validation
- Successor state generation
- DFS and BFS implementations
- Uniform Cost Search implementation
- A* Search with heuristic evaluation
- Performance analysis and solution comparison

## File Structure

- `solution_q1.py` – DFS and BFS implementations (my work)
- `solution_q2.py` – Uniform Cost Search implementation (my work)
- `solution_q3.py` – A* Search implementation with custom heuristics (my work)

## How to Run

```bash
python solution_q1.py
python solution_q2.py
python solution_q3.py
```
