Credit Card Default Prediction Analysis

A machine learning project analyzing credit card client data to predict default risk using feature engineering, feature selection, and decision tree classification.

Developed for CMPSC 432: Data Mining at Penn State.

Features
Data cleaning and preprocessing of real-world financial data
Feature ranking using Chi-Square and Mutual Information
Custom feature engineering with AVG_PAY_TIME
Min-Max normalization and one-hot encoding
Decision Tree classification with 5-fold cross-validation
Model comparison using reduced and full feature sets

Implemented Tasks
Data exploration and statistical analysis
Feature selection and ranking
Feature engineering and normalization
Decision Tree model training and evaluation
Cross-validation and performance comparison

File Structure
credit_card_data_analysis.ipynb – Analysis, preprocessing, and model development
credit_cards_2025-post.csv – Dataset used for training and evaluation
Credit Card Data Analysis Report.pdf – Written report and findings

How to Run
jupyter notebook credit_card_data_analysis.ipynb
