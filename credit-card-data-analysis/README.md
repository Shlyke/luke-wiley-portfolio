# Credit Card Data Analysis

A data mining and exploratory data analysis project on credit card default prediction using Python, pandas, and scikit-learn. The project includes data cleaning, feature engineering, statistical analysis, visualization, and predictive modeling.

Developed for CMPSC 432: Exploratory Data Mining at Penn State.

## Features

- Data cleaning and preprocessing of a real-world credit card dataset
- Exploratory data analysis with statistical summaries and visualizations
- Feature selection using Chi-Square and Mutual Information
- Feature engineering, normalization, and one-hot encoding
- Classification model evaluation using accuracy, precision, recall, and F1-score
- Comparison of baseline and feature-selected models

## Main Components

- Data loading and preprocessing
- Exploratory data analysis (EDA)
- Feature selection and ranking
- Feature engineering
- Model training and evaluation
- Performance comparison across feature subsets

## File Structure

- `credit_card_data_analysis.ipynb` – Complete analysis and modeling workflow (my work)
- `credit_cards_2025-post.csv` – Credit card default dataset
- `Credit Card Data Analysis Report.pdf` – Summary of methodology and results (my work)
